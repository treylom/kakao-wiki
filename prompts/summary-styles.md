# 요약 스타일 프리셋 (summary styles)

`summarize.py` 가 이 파일을 읽어 스타일별 프롬프트를 만든다. 각 스타일 = `## <id>` 헤더 +
바로 아래 코드블록(```)의 프롬프트 본문. `{context}` 자리표시자는 수집 데이터 요약(방·기간·
메시지 수·주요 발화자)으로 치환된다. 새 스타일을 추가하려면 같은 형식으로 한 절을 더 쓰면 된다.

> 환경 감지: `/prompt` 스킬이 있는 환경이면 `summarize.py` 가 그 스킬 사용을 안내하고,
> 없으면 아래 프리셋 중 선택한 스타일의 프롬프트를 그대로 출력한다(요약 LLM에 전달).

## brief
```
아래 카카오톡 대화({context})를 한국어로 **3줄 이내** 핵심만 요약해줘.
- 군더더기·인사말 빼고 결정/이슈/숫자 중심.
- 과장·평가 없이 사실만. 인용은 넣지 말 것.
```

## detailed
```
아래 카카오톡 대화({context})를 한국어로 **상세 요약**해줘.
- 주제별로 묶어 흐름이 보이게(무슨 일이 있었나 → 무엇을 정했나).
- 중요한 발언은 1~2개만 짧게 인용(따옴표), 인용은 원문 그대로.
- 마지막에 후속 액션이 있으면 항목으로 정리.
```

## bullets
```
아래 카카오톡 대화({context})를 한국어 **불릿 목록**으로 요약해줘.
- 한 줄에 한 포인트, 5~8개.
- 각 불릿은 명사형/간결체로. 이모지·마크다운 강조 없이 평문.
```

## formal
```
아래 카카오톡 대화({context})를 한국어 **보고체(존댓말)**로 요약해줘.
- "~했습니다 / ~로 결정되었습니다" 톤, 윗사람 보고 형식.
- 핵심 → 결정사항 → 후속 순서. 추측 없이 대화에 있는 내용만.
```

## casual
```
아래 카카오톡 대화({context})를 **편한 반말 톤**으로 짧게 풀어줘.
- 친구한테 "오늘 단톡방에서 이런 얘기 나왔어" 하듯이.
- 핵심만, 2~4문장. 과장 없이.
```
